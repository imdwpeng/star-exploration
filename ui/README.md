# 《星海迷航》UI设计系统

## 项目概述

本UI设计系统为《星海迷航》微信小游戏提供完整的界面设计和交互实现，基于科幻风格的低多边形+霓虹光效视觉风格。

## 设计理念

### 核心视觉风格
- **科技感**: 深空紫、星云蓝为主色调，赛博朋克粉、荧光绿为交互色
- **全息投影**: 半透明浮动面板，玻璃拟态效果
- **低多边形**: 简洁的几何形状，配合霓虹光效
- **沉浸体验**: 动态背景、粒子特效、流畅动画

### 设计原则
- **简洁直观**: 降低学习成本，3分钟内理解核心玩法
- **沉浸体验**: 科幻风格UI，配合空间音效
- **流畅反馈**: 每次操作都有即时视觉反馈
- **适配性**: 兼顾高中低端设备，保证流畅运行

## 核心界面

### 1. 首页 (Home Page)

**主要元素:**
- 状态栏: WiFi/电量/时间显示
- 玩家信息: 头像/昵称/等级
- 资源面板: 星币/水晶/体力
- 主视觉区域: 动态太空站预览
- 功能按钮: 开始探索/我的太空站/星际探索等
- 底部导航: 首页/活动/邮件/商店/设置

**交互特性:**
- 太空站360度旋转动画
- 资源面板悬停效果
- 按钮呼吸动画
- 底部导航状态指示

### 2. 游戏核心界面 (Game Core)

**主要元素:**
- 顶部信息栏: 关卡名/能量条
- 三维星图区域: 可交互的方块网格
- 待消除槽: 8个槽位，显示已选方块
- 倒计时进度条: 游戏时间显示
- 底部快捷操作: 回溯/扫描/重组

**交互特性:**
- 方块点击选中动画
- 奖励图案高亮显示
- 槽位填充和消除动画
- 资源获取飞行动画
- 扫描奖励组合功能

### 3. 太空站建设界面 (Station Build)

**主要元素:**
- 顶部标签页: 总览/建筑/科技/仓库
- 太空站预览: 可缩放/旋转的3D模型
- 建筑列表: 分类显示可建造建筑
- 建造面板: 详细属性和资源需求

**交互特性:**
- 建筑悬停预览
- 资源消耗显示
- 升级路径展示

### 4. 合作模式界面 (Co-op Room)

**主要元素:**
- 房间信息: 房主/模式/难度/人数
- 玩家列表: 头像/昵称/准备状态
- 任务分配面板: 区域划分
- 聊天区域: 快捷消息/文字输入
- 控制按钮: 准备/开始/离开

**交互特性:**
- 玩家状态指示
- 快捷消息按钮
- 实时聊天功能

## 动画系统

### 基础动画
- **方块点击**: 缩放+发光边框+轻微上浮 (150ms)
- **页面转场**: 淡入淡出 (300ms)
- **元素出现**: 滑入+淡入 (200-400ms)
- **消除特效**: 爆炸粒子+方块消失 (500ms)

### 特殊动画
- **太空站旋转**: 20s无限旋转
- **星云移动**: 30s无限旋转
- **方块浮动**: 6s无限浮动
- **倒计时**: 实时更新进度条

### 粒子特效
- 消除爆炸粒子
- 资源获取飞行动画
- 奖励提示闪烁

## 交互规范

### 操作反馈
| 操作 | 视觉反馈 | 音效 | 震动 |
|------|---------|------|------|
| 点击方块 | 缩放+发光 | 清脆点击声 | 轻微短震 |
| 槽内消除 | 爆炸粒子 | 爆破声+收集音 | 中等震动 |
| 获得资源 | 飞行动画 | 金币掉落声 | 无 |
| 游戏失败 | 屏幕变暗 | 低沉警告声 | 长震动 |
| 游戏胜利 | 烟花特效 | 胜利音乐 | 连续短震 |

### 状态处理
- **网络中断**: 浮动提示+自动重连
- **内存不足**: 降低画质+优化显示
- **电量过低**: 省电模式+特效关闭

## 性能优化

### 移动端优化
- **DrawCall**: ≤60 (使用InstancedMesh)
- **帧率**: ≥50fps (中端机)
- **内存**: <200MB (峰值)
- **启动**: <3秒 (首屏)

### 资源优化
- 使用SVG图标替代图片
- CSS动画替代JS动画
- 按需加载资源
- 对象池复用对象

## 响应式设计

### 屏幕适配
- 大屏手机 (≥6.5"): 两侧留白
- 中屏手机 (5.8"-6.4"): 全屏利用
- 小屏手机 (<5.8"): 紧凑布局
- 平板横屏: 左右分栏

### 性能分级
- **高端机**: 全特效, ≤60个方块
- **中端机**: 简化特效, ≤45个方块
- **低端机**: 基本特效, ≤30个方块

## 扩展开发

### 添加新界面
1. 在index.html中添加新页面div
2. 在styles.css中添加对应样式
3. 在game.js中添加交互逻辑

### 自定义主题
修改CSS变量:
```css
:root {
    --primary-color: #6366f1;
    --accent-color: #ec4899;
    --neon-cyan: #00bcd4;
    /* ... */
}
```
